3.1 Creating a New Document

Create a Eclipse project to which an XML document is to be added. Select File>New>Project in Eclipse.Select Java>Java Project in the New Project frame. Specify the Project Name, select the Create separate source and output folders, and click on the Next button in the New Java Project frame.

Figure 3.1.1 New Java Project

In the Java Settings frame click on the Finish button. A Java project gets added to Eclipse.

Figure 3.1.2 New Java Project

To add a XML, JSP, or JSF document to the project, select the project node in Eclipse and select File>New>Other.... In the New frame select XMLEspresso>XML, XSL, XML Schema... button. To create a DTD document select XMLEspresso>DTD. To create a CSS document select CSS 2.1.

Figure 3.1.3 New XML Document

In the Load XMLEspresso Eclipse Plugin version 6.0.0 frame select the plugin license. In the New XML Document or XML Schema frame select the project folder to which the XML document is to be added and specify a file name, ejb-jar.xml for an EJB 2.1 deployment descriptor. For the XML document to be based on a DTD or XML Schema in the DTD/Schema catalog select the Select XML Document type from catalog, including custom standalone documents radio button. To create a document from a DTD/XML Schema, select the Specify XML Schema or DTD for XML document radio button. Click on the Next button.

Figure 3.1.4 New XML Document or XML Schema

In the catalog frame select the type of XML, JSP, JSF document to be created. For example, select the J2EE 1.4>Ejb 2.1 type of an XML document. If the Target namespace from the XML Schema is to be applied to the document, select the Use Schema Target Namespace checkbox. Click on the Next button.

Figure 3.1.5 Document Catalog

The selected XML document gets added to the Eclipse project.

Figure 3.1.6 New XML Document
