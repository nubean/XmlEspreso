2.0 Configuring Plugin Preferences

Select Window>Preferences to configure plugin preferences.

Figure 2.0.1 Plugin Preferences

A Preferences. window is opened. Select the XMLEspresso XML Editor. Preferences that may be modified are validation with XML Schema, tag style, Internet proxy settings, and element style. Click in the radio button corresponding to the tag style to choose. Click on the Apply button. The selected style gets applied to the document.

Figure 2.0.2 Setting Style

To modify the foreground color, select the Element to which the setting is to be applied, Attribute for example, and click on the Change Foreground button.

Figure 2.0.3 Setting Foreground

In the Select Color frame select the foreground color and click on the OK button.

Figure 2.0.4 Selecting Foreground Color

In the Preferences frame click on the Apply button to apply the selected foreground to the selected element.

Figure 2.0.5 Applying Foreground Setting

To modify the background color, select the Element to which the setting is to be applied, End tag for example, and click on the Change Background button.

Figure 2.0.6 Setting Background

In the Select Color frame select the background color and click on the OK button.

Figure 2.0.7

In the Preferences frame click on the Apply button to apply the selected background to the selected element.

Figure 2.0.8 Applying Background Setting

Close and re-open the document. The Preferences settings get applied to the XML documents.

Figure 2.0.9 Preferences Settings

To add XML Schema validation to an XML document, select the Validate Schema checkbox and click on the Apply button.

Figure 2.10 Selecting Validate Schema

A XML Schema tab gets added for the schema for the document.

Figure 2.11 XML Schema Tab

The XML Schema validates the XML document. If the document has an error, the error gets displayed.

Figure 2.12 Validation Errors

To set the foreground and background color of elements in a CSS document select the CSS Editor.

Figure 2.13

To set the foreground and background color of elements in a DTD docoument select the DTD Editor.
