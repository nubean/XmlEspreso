3.3 Adding a Document

Select the folder in which an XML document is to be opened. Select File>Import... to open a document.

Figure 3.3.1 File>Import

In the Import frame select File System and click on Next. In the File System frame select the XML document to import and click on the Finish button. The selected document gets added to the Eclipse project. To open the added document, right-click on the document and select Open With>XMLEspresso XML Editor

Figure 3.3.2 Imported Document

The added document gets opened in Eclipse. The XML document nodes get displayed.
