5.1 Adding an Element to a Document

To add an element in Design mode select the preceding node corresponding to the element in the document panel. For example select the 'enterprise-beans' node to add an 'entity' node. Click on the Design tab.

Figure 5.1.1 Selecting the Design Tab

A node panel displays nodes corresponding to the sub-elements of the selected element.To add a sub-element tag right-click on the node corresponding to the element and click on insert. For example to add a 'entity' element in the 'enterprise-beans' element right-click on the 'entity' node in the node panel and click on Insert.

Figure 5.1.2 Adding an Element

The selected element gets added to the document. A node corresponding to the selected element gets added to the Outline view.

Figure 5.1.3 Document with the Added Element Tag

To add an attribute to an element select the element in the Outline view. Select the Design tab.

Figure 5.1.4 Selecting an Element to add an Attribute

In the Design frame, in the properties sheet specify the values for attribute/s. Click on Enter.

Figure 5.1.5 Properties sheet

The specified attributes get added in the XML document. An attribute node gets added to the Outline view.

Figure 5.1.6 XML Document with the added Attribute
