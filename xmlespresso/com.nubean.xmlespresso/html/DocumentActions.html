3.4 Document Actions

XMLEspresso plugin provides various document actions, which may be accessed by right-clicking on the vertical bar between the document and the Package Explorer. Document mode may be toggled between Text Mode (the default) and XML Mode. To refresh the document select Refresh. Validate a document with a DTD with Validate using Dtd. Validate a document with an XML Schema with Validate using Schema. Save the XML Schema for an XML document with Save Schema As.

Figure 3.4.1 Toggling Document Mode

For content assist, position the cursor in the document where content assist is required and select Content assist.

Figure 3.4.2 Content Assist

A list of elements or attributes that may be added gets displayed. Select an element or an attribute and select Enter to add the element/attribute.
