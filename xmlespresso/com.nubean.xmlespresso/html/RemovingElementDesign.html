5.2 Removing an Element from a Document

To remove an XML element, select the element containing the element to be removed in the Outline view. For example to remove the 'entity' element in a document of type J2EE>Ejb 2.1, select 'enterprise-beans' element in the Outline view.

Figure 5.2.1 Selecting an Element from which an element is to be removed

Select the Design tab. Right-click on the node corresponding to the element to remove in the Design frame node panel. For example select the 'entity' element. Select Remove in the pop-up menu.

Figure 5.2.2 Removing an Element

The selected element gets removed from the Design frame node panel. Select the Source tab. The selected element gets removed from the XML document and the Outline view. Similarly, to remove an element attribute select the element which has the attribute and select the Design tab. In the Design frame right-click on the attribute and select Remove.
