XMLEspresso 8.0.0

What is new in version 8.0.0?

The following features are new in version 8.0.0:

  • Version 8.0.0 offers improved Design mode whereby new elements can be added by right clicking on elements in the Design Tree.
  • Design Mode juxtaposes Design Tree, Element Schema, Element Tree and Element Attributes for a much more intuitive interface.
  • Improved XPath Search

Key Features

XMLEspresso 8.0.0 is a full featured editor for creating and maintaining XML documents. Its important features include:
  • It supports stand alone, Document Type Definition (DTD) and W3C XML Schema based documents.
  • To simplify the creation of commonly used XML documents, there is a built-in, extensible catalog of XML grammars.
  • Creation of an XML document is strongly guided by the underlying DTD or W3C XM Schema and the editor makes a continual effort to maintain the document as a well-formed XML document.
  • Every XML document when opened has a Design view and a Source view. Both views are completely integrated and observe and react to changes in either view.
  • Each XML document can be validated against its DTD or W3C XML Schema.
  • An XML document can be saved as a PDF document.
  • An XML DTD can be saved as an XML Schema.
  • XML document can be searched for XPath expressions.
  • XML document can be transformed using XSLT transformations.
  • It supports context sensitive content assist using CTRL + Space in Source view for inserting sub-element tags and for defining attributes.
  • When Text Mode is toggled off, direct editing of the document is disabled, but Source view content assist is still available
