3.3 Adding a Document

Add a document to a project with the File>Add Document.. menu item.

Figure 3.3.1 File>Add Document

To add a document to a project from the toolbar select the Add Document icon

A Add Document window opens. Select the document to add and click the Add document To Project button.

Figure 3.3.2 Add Document Window

The selected document gets added to the project.
