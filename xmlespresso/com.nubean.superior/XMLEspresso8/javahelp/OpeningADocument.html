3.2 Opening a Document

Select File>Open Document... to open a document.

Figure 3.2.1 File>Open Document

To open a document from the toolbar click on the Open Document icon .

A Open Document window opens. Select the document to open and Click the Open Document button.

Figure 3.2.2 Open Document Window

If a DTD or XML Schema is not specified in the XML document, a Schema is automatically created on the fly.

The selected document gets opened.

Figure 3.2.4 Opened Document
