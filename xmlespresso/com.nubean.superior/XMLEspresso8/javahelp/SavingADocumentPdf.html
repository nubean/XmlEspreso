3.6 Saving a Document as a PDF Document

Right-click in an open document. A pop-up menu opens up. Select Save As PDF... in the pop-up menu.

Figure 3.6.1 File>Save As PDF

A Save As PDF window opens up. Specify a file name for the PDF document and click the Save Document AS PDF button.The XML document gets saved as a PDF document.
