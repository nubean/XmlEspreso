4.3 Editing a Document in Text Mode

A document opens in text mode by default. Open the document pop-up menu; Toggle Text Mode is selected by default. Edit the document in text mode. In text mode elements may be modified as text and a document may not be well formed. The Xml Check view displays the errors in the document.

Figure 4.3.1 Toggle Text Mode

In the text mode, uncheck Toggle Text Mode in the document pop-up menu.

Figure 4.3.2 Toggle to XML Mode

The document opens in XML mode. In XML mode, element text may be modified, but the elements do not get modified as text.
