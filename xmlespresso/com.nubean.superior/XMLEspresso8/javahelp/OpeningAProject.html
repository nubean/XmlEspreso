2.2 Opening a Project

Select Open Project... in the File menu to open a project.

Figure 2.2.1 File>Open Project

A project is also opened with the Open Project icon in the Tool bar.

Save the project which is already open.

Figure 2.2.2 Save Project Prompt

A Open Project window opens up. Select the project to open and click the Open Project button.

Figure 2.2.3 Open Project Window

The selected project opens in a panel.
