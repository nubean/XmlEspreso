3.5 Saving a Document Schema

Right-click in a document. A pop-up menu opens up. Select Save Schema As... menu item to save schema corresponding to the DTD for the XML document.

Figure 3.5.1 Save Schema As...

A Save Schema As window opens. Specify a schema file name and click the Save Schema As button. A Xml Schema corresponding to the document DTD is generated and saved in the specified file.
