2.1 Creating a New Project

Select New or New Project in the File menu to create a new project.

Figure 2.1.1 File>New Project

A new project is also created with the New icon in the tool bar.

If New is selected in the File menu or the New icon is selected, a Select New window opens up. Select Project from the choice list.

Figure 2.1.2 Select New Window

Close current project before opening a new project.

Figure 2.1.3 Close Current Project Window

When New Project is selected in the File menu or Project is selected in the Select New window a Project Wizard is displayed. The Project Wizard is used to create a new project. In the Project Wizard the General tab is selected by default. Specify the project name in the Name field, the project directory in the Directory field and project description in the Description field. Click the Next button.

Figure 2.1.4 Project Wizard

The Paths tab gets selected in the Project Wizard. Specify the output directory path in the Output Path field, the backup directory path in the Backup Path field and the working directory in the Working Directory field. Click the Next button.

Figure 2.1.5 Paths Tab Frame in Project Wizard

The Settings tab gets selected in the Project Wizard. Specify the encoding in the Encoding choice list. Specify the project title in the Project Title field, the project author in the Author field, the project version in the Version field and copyright in the Copyright field. Click Finish to finish the project.

Figure 2.1.6 Settings Tab Frame in Project Wizard

A new project gets created and added to the node panel.
