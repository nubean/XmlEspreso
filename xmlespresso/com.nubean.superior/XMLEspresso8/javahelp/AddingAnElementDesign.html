5.1 Adding an Element to a Document

If an XML document is well-formed the outline for the document gets displayed in the Outline view. To edit a element in Design mode select the node corresponding to the element in the document outline. For example select the 'enterprise-beans' node in a document of type JavaEE 5>Ejb 3.0.

Figure 5.1.1 Node Panel

Click on the Design tab. The Design tab frame is displayed in the foreground. A node panel displays nodes corresponding to the sub-elements of the selected element. To add a sub-element tag right-click on the node corresponding to the element and click on Insert. For example to add a 'entity' element in the 'enterprise-beans' element right-click on the 'entity' node in the node panel and click on Insert.

Figure 5.1.2 Design Frame Node Panel

The selected element gets added to the document, and you can switch back to the Source view to see it added to the source.

Figure 5.1.3 Document with the Added Element Tag

A node corresponding to the selected element gets added to the document outline.

Figure 5.1.4 Document Outline with the Added Element Node

For an element with attributes, when an element is selected in the document outline and the Design tab selected, attribute list gets displayed. For example, when the 'enterprise-beans' element in a document of type JavaEE 5>Ejb 3.0 is selected, an id attribute field is displayed. Enter the attribute value in the id attribute field and press the Enter button.

Figure 5.1.5 Specifying Attribute Value

The attribute value gets added to the 'enterprise-beans' element in the document.

Figure 5.1.6 Document with the Added Attribute

A attribute node gets added to the document outline.

Figure 5.1.7 Document with the Added Attribute

To add text to a text element, select the element in the document outline. For example, in a document of type JavaEE 5>Ejb 3.0, add 'ejb-name' to 'entity' element and select the 'ejb-name' element in the document outline. Select the Design tab.

Figure 5.1.8 Text Element Node

A text area for the element tag is displayed. Specify the element text and press the Apply button. To add a CDATA section select the CDATA checkbox, specify the text and click on Apply. Click on the OK button.

Figure 5.1.9 Element Text Area

The text gets added to the element in the document.
Figure 5.1.10 Document with Element Text

A text node gets added in the document outline.
