3.4 Saving a Document

Select the document tab corresponding the document to save. Save the document with File>Save Document.

Figure 3.4.1 File>Save

To save a open document from the tool bar click on the Save icon.

To save a document from the Project node pop-up menu, right-click on the document node corresponding to the document to save and select Save in the pop-up menu.

Figure 3.4.2 File>Save

Save a document with a specific file name with File>Save Document As....

Figure 3.4.3 File>Save Document As

To save a document with a specific file name from the Project node pop-up menu, right-click on the node corresponding to the document to save and select Save As... in the pop-up menu.

Figure 3.4.4 File>Save Document As

A Save As window opens.Specify a file name and click on the Save Document As button.

Figure 3.4.5 Save As Window

To save a open document with a specific file name from the tool bar click on the Save As icon.

Save selected documents with File>Save Documents....

Figure 3.4.6 File>Save Documents

A Save Documents window opens up. Select a document to save with a click in the checkbox next to the document. Multiple documents may be selected with Ctrl+Shift. Select the All button to save all the documents and select the None button to select none of the documents.

Figure 3.4.7 Save Documents Window

Save all open documents with File>Save All.

Figure 3.4.8 File>Save All

To save all open documents from the tool bar click on the Save All icon.
