2.1 Introduction

A project consists of a group of related documents, governed by a common set of project properties. For example, a project could contain documents related to a J2EE application. Such a project may contain XML deployment descriptor documents such as an ejb-jar.xml, a weblogic-ejb-jar.xml and a weblogic-cmp-rdbms-jar.xml.

Common project properties are configured with Project Properties menu item in the Tools menu.
