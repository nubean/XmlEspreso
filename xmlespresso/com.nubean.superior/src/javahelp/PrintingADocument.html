3.7 Printing a Document

To print a document select File>Print.... To print a document from the toolbar click on the Print icon .

Figure 3.7.1 File>Print

A Print window opens up. The General tab is selected by default. Specify the Print Service, the Print Range and the Copies box values.

Figure 3.7.2 Print window

Select the Page Setup tab to specify the page setup.

Figure 3.7.3 Page Setup

Select the Appearance tab to specify print output appearance. Click the Print button to print.

Figure 3.7.4 Appearance

Click the Print button to print.
