3.8 Transforming a Document using an XSLT File

Open the pop-up menu in an XML document. Select Xsl Transform... menu item to transform an XML document using an XSLT document.

Figure 3.8.1 Xsl Transform...

A Open Xslt File opens up. Specify an XSLT file.

Figure 3.8.2 Open Xslt File

A Save Transform Output window opens up. Specify an output file name. Click the Save Transform Output button. The XML document is transformed using the specified XSLT file to the specified output file.

Figure 3.8.3 Save Transform Output
