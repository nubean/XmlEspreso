3.9 Removing a Document

Select the document to remove in the project panel.

Figure 3.9.1 Project Panel with a Selected Document

Remove the selected document from a project with the File>Remove Document... menu item.

Figure 3.9.2 File>Remove Document

To remove a document with the toolbar, select Remove Document icon in the toolbar.

To remove a document with the Project node pop-up menu, right-click on the document node corresponding to the document to remove and select the Remove menu item.

Figure 3.9.3 Project Node pop-up menu>Remove

The selected document gets removed from the project. A document may also be deleted from the project.

Figure 3.9.4 Project Panel with a Document Removed
